# Portfolio of knt5

https://knt5.github.io/

```
# Move to the vue project directory
cd portfolio

# Setup
npm i

# Compiles and hot-reloads for development
npm run serve

# Compiles and minifies for production
npm run build

# Lints and fixes files
npm run lint
```
